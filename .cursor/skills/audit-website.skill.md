# Website Audit Best Practices

Comprehensive guidelines for auditing websites across multiple dimensions.

## When to Apply

- Performing site-wide audits
- Reviewing performance metrics
- Checking accessibility compliance
- Analyzing SEO factors
- Reviewing security practices
- Checking mobile responsiveness

## Key Principles

- Performance: Core Web Vitals, load times, bundle sizes
- Accessibility: WCAG compliance, keyboard navigation, screen readers
- SEO: Meta tags, structured data, sitemaps
- Security: HTTPS, CSP headers, XSS protection
- Mobile: Responsive design, touch targets, viewport settings
- Best Practices: Semantic HTML, proper heading hierarchy, alt text
