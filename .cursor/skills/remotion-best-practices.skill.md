# Remotion Best Practices

Guidelines for creating performant and maintainable Remotion video compositions.

## When to Apply

- Creating video compositions
- Optimizing video rendering performance
- Working with animations in Remotion
- Managing video assets and media

## Key Principles

- Use composition-level caching for expensive computations
- Optimize asset loading and preloading
- Minimize re-renders during video playback
- Use appropriate frame rates and durations
- Handle memory efficiently for long videos
