# 飞书文档 — 参考

## 常见导出格式

- **Markdown**：飞书文档可导出为 .md，标题、列表、表格会保留，便于直接粘贴到对话。
- **复制到剪贴板**：在飞书里选中内容复制，粘贴到 Cursor 对话即可。

## 文档链接格式

- 文档：`https://*.feishu.cn/docx/...` 或 `https://*.larkoffice.com/docx/...`
- 多维表格：`https://*.feishu.cn/sheets/...` 或 `.../base/...`

Agent 无法直接打开上述链接，需用户粘贴正文或导出内容。

## 你可以在本文件补充

- 项目内常用的飞书文档名称或链接（仅作备注，不写敏感信息）。
- 团队对「需求文档」「接口说明」的章节约定。
- 飞书表格与代码字段的对应关系（如「文档列 A = 接口字段 xxx」）。
