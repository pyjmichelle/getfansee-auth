# Building Native UI

Guidelines for creating native-feeling user interfaces on the web.

## When to Apply

- Building mobile-responsive components
- Implementing touch interactions
- Creating native-like animations
- Designing gesture-based interfaces
- Optimizing for mobile performance

## Key Principles

- Touch-friendly target sizes (min 44x44px)
- Smooth scroll and momentum
- Native-feeling transitions
- Proper viewport handling
- Gesture recognition patterns
