## 1. 产品概述

### 1.1 产品定位

**GetFanSee** 是一个创作者经济平台，连接创作者（Creator）和粉丝（Fan），支持：

- 订阅制内容（Subscription）
- 单次付费内容（Pay Per View / PPV）
- 社交互动（点赞、评论、分享）

### 1.2 用户角色

| 角色                  | 描述                     | 核心功能                                  |
| --------------------- | ------------------------ | ----------------------------------------- |
| **Fan（粉丝）**       | 默认角色，浏览和消费内容 | 浏览 Feed、订阅创作者、购买 PPV、评论互动 |
| **Creator（创作者）** | 升级角色，需 KYC 认证    | 发布内容、设置定价、查看数据、管理收益    |
| **Admin（管理员）**   | 后台管理角色             | 内容审核、KYC 审核、举报处理              |

### 1.3 核心业务流程

```
粉丝注册 → 浏览内容 → 订阅/购买 → 解锁内容 → 互动
    ↓
升级为创作者 → KYC 认证 → 发布内容 → 获得收益 → 提现
```

### 4.1 认证系统

#### 登录流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  输入邮箱   │───▶│  输入密码   │───▶│  验证身份   │
│  密码表单   │    │  点击登录   │    │  检查封禁   │
└─────────────┘    └─────────────┘    └──────┬──────┘
                                             │
                   ┌─────────────┐           │
                   │  跳转首页   │◀──────────┘
                   │  /home      │
                   └─────────────┘
```

**登录页面元素**：

- Logo + 产品名
- Tab 切换（登录/注册）
- 邮箱输入框
- 密码输入框（带显示/隐藏切换）
- "记住我" 复选框（可选）
- 登录按钮（主渐变按钮）
- "忘记密码" 链接
- OAuth 登录按钮（Google）- 测试模式禁用

#### 注册流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  输入邮箱   │───▶│  输入密码   │───▶│  确认 18+   │
│  密码表单   │    │  确认密码   │    │  年龄验证   │
└─────────────┘    └─────────────┘    └──────┬──────┘
                                             │
                   ┌─────────────┐           │
                   │  发送验证   │◀──────────┘
                   │  邮件       │
                   └──────┬──────┘
                          │
                   ┌──────▼──────┐
                   │  验证成功   │
                   │  跳转首页   │
                   └─────────────┘
```

**注册页面元素**：

- 邮箱输入框
- 密码输入框（至少 8 字符）
- 确认密码输入框
- 18+ 年龄确认复选框（必选）
- 服务条款同意复选框
- 注册按钮
- 已有账号？登录链接

#### 邮箱验证页

**状态**：

1. **验证中** - 显示加载动画
2. **验证成功** - 显示成功图标，自动跳转
3. **验证失败** - 显示错误信息，重发按钮

| `subscribe-gradient` | 订阅按钮 | 紫色渐变 + 发光          |
| `unlock-gradient`    | 解锁按钮 | 金色渐变 + 发光          |
| `success-gradient`   | 成功状态 | 绿色渐变                 |

#### 尺寸 (Sizes)

| 尺寸      | 高度    | 最小触摸区域 | 用途       |
| --------- | ------- | ------------ | ---------- |

| `icon-lg` | 48x48px | 48x48px      | 大图标按钮 |

- **按下**: `scale(0.98)`

## 9. 无障碍设计

### 9.1 颜色对比

- 文字与背景对比度：至少 4.5:1 (WCAG AA)
- 大文字对比度：至少 3:1
- 交互元素：清晰的焦点指示器

### 9.2 键盘导航

- 所有交互元素可通过 Tab 键访问
- 支持 Enter/Space 激活按钮
- 支持 Escape 关闭弹窗
- 焦点顺序符合视觉顺序

### 9.3 屏幕阅读器

- 所有图片有 `alt` 文本
- 图标按钮有 `aria-label`
- 表单元素有关联的 `label`
- 动态内容有 `aria-live` 区域

### 9.4 动效偏好

- 支持 `prefers-reduced-motion`
- 减少动效模式下禁用非必要动画
